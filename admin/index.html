連結 GitHub

後台需要一組 GitHub Personal Access Token 才能把文章寫入網站。Token 只存在你的瀏覽器裡，不會上傳到任何地方。

怎麼取得 Token：
1. 點這個連結 → GitHub → Settings → Fine-grained tokens → Generate new token
2. Repository access 選 Only select repositories，選 claudia-fox.github.io
3. Permissions → Repository permissions → Contents 設為 Read and write
4. 按 Generate token，複製後貼到下方
